Variância

Para calcularmos a variância, devemos somar a média subtraída de cada valor e elevar a quadrado e dividir o valor final pela quantia de valores.

Usando as notas de antes: 8, 4.5, 8,5.

((8 - 7)² + (4,5 - 7)² + (8,5 - 7)²) / 3 = ((1)² + (-2,5)² + (1,5)²) / 3 = (1 + 6,25 + 2,25) / 3 = 9,5 / 3 ≈ 3,17

Temos que a variância dessas notas é igual a 3,17.

Observação: Caso os valores sejam altura em metros, por exemplo, já que a variância eleva os valores ao quadrado, o resultado da variância seria em m².

Desvio padrão

O desvio padrão, significa o quanto as notas desviaram da média, quando se tem a variância, é muito simples: Basta tirar a raíz quadrada da variância.

√3,17 ≈ 1,78

Observação: Novamente, caso os valores fossem altura em metros, a variância seria em m², e já que tiramos a raiz da variância, o valor é em m.

Coeficiente de variação

Para calcular o coeficiente de variação, que é o quanto os dados estão dispersos à média, basta fazer o desvio padrão dividido pela média e depois multiplicar por 100.

1,78 / 7 * 100 ≈ 0.2543 * 100 ≈ 25.43%

Planilhas Eletrônicas

Algo que também foi trabalhado em aula foi o uso das planilhas eletrônicas.

SOMA() para somar

MEDIA() para calcular média

MODE() para calcular a moda

MEDIAN() para calcular a mediana

VAR.P() para calcular a variância da população inteira.

STDEV.P() para calcular o desvio padrão da população inteira

Existem diversas outras funçôes matemáticas em planilhas eletrônicas, o que é bem conveniente quando se tem diversos valores e/ou vários valores.
